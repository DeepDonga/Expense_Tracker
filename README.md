#  Expense Tracker

##  Overview

**Expense Tracker** is a web-based financial management application designed to simplify and digitalize the process of managing personal finances. The platform allows users to record income and expenses, categorize transactions, manage multiple accounts, and visualize financial data through interactive charts.

The project falls under the **FinTech domain**, aiming to provide convenience, accessibility, and an enhanced user experience for tracking personal financial activities.

---

##  Objectives

The main objectives of the Expense Tracker system are:

- To provide users with an easy-to-use platform for managing income and expenses.
- To allow classification of transactions into meaningful categories such as Food, Rent, and Clothes.
- To design a lightweight, responsive web-based application accessible across devices.
- To help users analyze financial habits through visual insights.
- To replace manual financial tracking methods with a digital and efficient solution.

---

##  Technology Stack

### Front-End Development

- **HTML & CSS** – Used to structure and design the user interface.
- **JavaScript (ES6+)** – Adds interactivity, including form validation and dynamic chart rendering.
- **Bootstrap** – A responsive CSS framework used to design mobile-first and user-friendly layouts.

### Back-End Development

- **Flask** – A lightweight Python web framework used to handle routing and server-side logic.
- **MongoDB** – A NoSQL database used to store user data, transactions, accounts, and categories in flexible JSON-like documents.

---

##  Application Workflow

1. User registers or logs in.
2. User selects "Add Transaction".
3. Enters transaction details:
   - Select account
   - Select category
   - Enter amount and type
   - Submit transaction
4. Database updates automatically.
5. Dashboard charts and balance update dynamically.
